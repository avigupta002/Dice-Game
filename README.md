This is a simple **two-player dice game** built using **HTML, CSS, and JavaScript**.

## Game Rules
- The game is played by **two players**, one after the other.
- Each player rolls the dice on their turn.
- Players accumulate points based on the dice value.
- A **target score** is set at the start of the game.
- The **first player to reach the target score wins** the game.

## Features
- Turn-based gameplay
- Random dice roll
- Simple and clean UI
- Winner announcement when target score is achieved

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Open the game in a browser.
2. Set the target score.
3. Players take turns rolling the dice.
4. The player who reaches the target score first wins.

This project is created for learning and practicing basic web development concepts.
